# Mockup Comparison — parent dashboard

## v1 — DJ 헤더 + 자녀 chip 좌측 (✅ 선택)

```
[김민수 (3-1) ▼]   [🌟] [+ 자녀] [👤]
─────────────────────────────────────
📚 김민수의 작품
[카드들]
🌟 학급 자랑해요
[chips]
```

장점: student-portfolio + DJ 보드 동일 헤더 패턴 일관. 신규 학습 0.
단점: 자녀 ≥2명일 때 chip 폭 길어지면 우측 액션과 충돌 가능 — flex-wrap 으로 완화.

## v2 — 좌측 사이드바 자녀 리스트 ❌

학생 portfolio 의 좌측 학생 리스트 패턴 차용. 단점: 학부모는 자녀 1~3명이 일반이라 사이드바 reservation 비용 높음. 모바일 stack 필수.

## v3 — Netflix 프로필 진입 ❌

로그인 후 자녀 카드들 그리드 → 클릭 시 그 자녀 portfolio. 단점: 매번 진입 + 1 click 추가. 자녀 1명일 때 불필요.

## v4 — 단일 자녀 자동 + 다자녀만 모달 셀렉터 ❌

자녀 1명이면 바로 portfolio. ≥2명만 첫 진입 시 자녀 선택 모달. 단점: 자녀 전환 매번 모달 — 셀렉터 chip 보다 인터랙션 비용 높음.
