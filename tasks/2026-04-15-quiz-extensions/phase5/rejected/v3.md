## QuizReportModal
v1 과 동일.

## 특징
- 각 문항은 접힌 상태 (질문 한 줄 + 정답 힌트 + 삭제).
- 클릭 시 펼쳐서 편집. 한 번에 한 개만 펼쳐짐 (accordion).
- 펼쳐진 문항은 v1 세로 stack 동일.

## 장점
- 10문항도 한 화면에 개요 표시. 네비게이션 최강.
- "어떤 문항을 수정해야 하나" 탐색이 쉬움.

## 단점
- 여러 문항 동시 편집 불가능 → 빠른 일괄 편집에 불리.
- 새 문항 추가 시 자동 펼침 필요 (UX 세부 결정 많음).
- 실수로 접힌 상태에서 저장 시도하면 어떤 카드가 빈지 시각 힌트 약함.
- 구현 복잡도 상승 (accordion 상태 관리).
