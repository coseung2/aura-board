## 결정 근거

**v1 Classic Sidebar를 기본으로 선택**, 반응형 breakpoint 768px 이하에서는 v3 Mobile Tabs로 전환.

- **교육 맥락**: 초등 5학년은 "지금 어디까지 왔는가"를 명확히 알아야 함. v1의 좌측 스텝퍼는 11단계 전체를 한눈에 보여줌.
- **상태 인지**: ⬜🟡🟢🔵✅ 아이콘 + "수정 중"/"승인 완료" 텍스트가 동시에 노출되어 상태 인지가 가장 빠름.
- **브레이크아웃룸 연계**: 기존 breakout 보드도 좌측 사이드바 없이 캔버스 중심이나, 미션 보드는 "진행 관리"가 핵심이므로 사이드바가 자연스러움.
- **모바일 대응**: 768px 이하에서 사이드바는 하단 고정 탭(v3)으로 전환. 이는 기존 `SidePanel` 모바일 동작과 일관됨.

## 반응형 전환 규칙

| 뷰포트 | 레이아웃 |
|---|---|
| ≥ 1024px | v1 Classic Sidebar (280px 고정) |
| 768px ~ 1023px | v1 Sidebar 축소 (220px) 또는 v2 Rail로 자동 전환 |
| < 768px | v3 Mobile Tabs (상단 가로 스크롤 탭) |
